# The CupCaitery

Custom cookies and baked goods website for a Northern Kentucky home bakery.

## About

The CupCaitery is run by Caite, a Certified Pastry Chef and Sullivan University graduate specializing in custom sugar cookies, cupcakes, cakes, and other sweet treats for birthdays, showers, graduations, weddings, and more.

## Project Structure

## Features

- Single-page responsive design
- Mobile-friendly navigation
- Animated sprinkles background
- Menu/pricing for all treats
- Social media integration (Instagram & Facebook)
- About section with baker bio

## Menu Items

- Custom Sugar Cookies
- Logo Cookies
- Cakes
- Cake Pops
- Cupcakes
- Custom Cookie Cutters
